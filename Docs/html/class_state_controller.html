Detailed Description

This controller object contains state information and acts as a go between the view and model.

It receives commands from user interaction and via the network and performs actions accordingly. It has a number of signals that UIs should connect to in order to receive information.

Member Enumeration Documentation

enum StateController::VoIPState

Possible VoIP states for the application to be in.

A List of enumerated states that the application can possibly be in. This is used to represent a finite-state machine.

Enumerator:
Ready 

Ready state shows that the application can receive calls.

Calling 

The application is waiting on calling another user.

InCall 

The application is in a call with another user.
