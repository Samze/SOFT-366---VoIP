Detailed Description

Discovers local peers on the network to communicate with.

This class broadcasts and receives a beacon signal that is used to determine who is avaiable on the network for calling. A signal is sent out periodically to show an "alive" state. The same port is binded to listen to other peers, they are added to a list. Periodically a check is done to see the last know time of an update. This is used to remove peers from the list. This class inherits from QUdpSocket.
